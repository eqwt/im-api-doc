# ------API------

## 注册

```
扩展字段：json字符串
{
	username：账号
	password：登录密码
	safe_pwd：安全密码
}

该用户比较特殊，只用于注册，限制登录
响应示例：
{
  "errCode": 1, // 注册固定返回 1，根据responseCode判断是否成功
  "responseCode": 状态码
  
  // 状态码定义
  20000：注册成功
  20001：账号不能为空
  20002：账号必须6-20位
  20003：账号格式不正确
  20004：账号已存在
  20005：登录密码不能为空
  20006：登录密码必须8-20位
  20007：登录密码格式不正确
  20008：安全密码不能为空
  20009：安全密码必须是8位数字
  20010：安全密码不能是相同数字
  20011：安全密码不能是连续数字
  20099：服务异常
}
```

## 登录

```
请求扩展字段(在需要二次验证时)：
{
	verify_way：验证方式(1=原设备验证 2=安全密码验证)
	
	// verify_way=1时，
	code：验证码，该字段为空时表示发送至原设备，不空时表示验证验证
	
	// verify_way=2时
	safe_pwd：安全密码
}

响应：
{
  "errCode": 错误码(0=成功 1=失败)
  "responseCode": 状态码
  
  // 状态码定义
  20001：用户不存在
  20002：密码错误
  20003：非信任设备
  20004：验证方法非法
  20005：原设备验证码发送成功
  20006：原设备验证码不正确
  20007：安全密码不正确
}
```
